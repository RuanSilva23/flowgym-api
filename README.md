# 🏋️‍♂️ FlowGym

> API REST para monitoramento de treinos, controle de cargas e evolução física.

---

## 📝 Sobre o Projeto

O **FlowGym** é uma aplicação backend desenvolvida para funcionar como um assistente digital de treinos. O objetivo principal é permitir que praticantes de atividades físicas registrem seus treinos, monitorem sua evolução em exercícios (como histórico de cargas e repetições) e gerenciem suas rotas de atividades de forma inteligente.

*Nota: Este projeto está em desenvolvimento ativo.

---

## 🚀 Tecnologias e Ferramentas

O ecossistema do projeto foi escolhido com foco em performance, tipagem forte e padrões de mercado para aplicações corporativas:

- **Linguagem:** Java 25
- **Framework:** Spring Boot 3
- **Persistência de Dados:** Spring Data JPA/ Hibernate
- **Banco de Dados:** PostgreSQL
- **Gerenciador de Dependências:** Maven
